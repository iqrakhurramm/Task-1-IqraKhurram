# Rule-Based Library Chatbot

A simple, deterministic console chatbot that simulates basic human interaction using an efficient dictionary lookup instead of complex conditional statements.

## Features
- Runs in a continuous user execution loop.
- Sanitizes input by automatically handling uppercase letters and extra whitespace.
- Uses a hash map/dictionary lookup for constant time O(1) performance.
- Includes a fallback response for unknown commands.

## Files
- `chatbot.py`: The main program file containing the script logic.

## Usage Examples
- Type 'hello' or 'hi' for a greeting.
- Type 'borrow book' to see checkout options.
- Type 'hours', 'location', or 'wifi' for basic information.
- Type 'exit' or 'bye' to close the program.
